# 🛒 Tienda TRIDADO

Tienda online simple para vender productos y enviar pedidos directamente por WhatsApp.

## 🚀 Funciones

- Ver productos
- Agregar al carrito
- Calcular total
- Enviar pedido directo por WhatsApp

## 📦 Productos

- Generala
- 4 en Raya
- Dados

## 📱 Cómo funciona

1. El usuario entra a la web
2. Agrega productos al carrito
3. Presiona **Enviar pedido por WhatsApp**
4. Se abre WhatsApp con el pedido listo para enviar
